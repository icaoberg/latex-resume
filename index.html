LaTeX Résumé Template

When compared to the result of any word processing application, using LaTeX gives you a significantly more professional-looking document. When creating a document as important as your CV or your résumé, a professional result becomes even more important.

The following sample PDF shows the result of using this document class, with source included to see how to obtain that output. The document class is based heavily on the sample document from Seth Holloway, with some ideas taken from cv-template.

This résumé class requires the Fontin font, which can currently be downloaded and used freely. Please check the font license though, as sometimes licenses may change without warning. If you find that you need to use a different font, add this line before beginning the document:

\setmainfont[SmallCapsFont=Verdana]{Verdana}

Replace "Verdana" with your desired font. You can also specify a different font for your small caps font if needed; not every font provides a small caps font set. The Fontin small caps font is set to "Fontin SmallCaps".

The résumé class uses some features that are not available in standard LaTeX; to compile documents using this résumé class, use "xelatex" instead.
